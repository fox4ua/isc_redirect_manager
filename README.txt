ISC Redirect Manager
====================

Модуль для керування правилами переадресації матеріалів і термінів таксономії в Drupal 10/11.

Основні можливості
------------------
- окремі адміністративні розділи для матеріалів і словників;
- перегляд наборів із переходом до правил конкретного типу матеріалу або словника;
- створення, редагування, видалення, увімкнення та вимкнення правил;
- упорядкування правил drag & drop в межах одного набору;
- перевірка правил у порядку зростання ваги;
- спрацьовування лише першого активного правила, що збіглося;
- підтримка внутрішніх шляхів призначення з автодоповненням;
- окремі налаштування модуля.

Адміністративні сторінки
------------------------
- огляд модуля: /admin/config/search/isc-redirects
- матеріали: /admin/config/search/isc-redirects/materials
- словники: /admin/config/search/isc-redirects/vocabularies
- правила конкретного набору: /admin/config/search/isc-redirects/{entity_type}/{bundle}
- налаштування: /admin/config/search/isc-redirects/settings

Хлібні крихти
-------------
Модуль формує власні хлібні крихти через hook_system_breadcrumb_alter().
Фінальна схема:
- Правила переадресації -> Матеріали
- Правила переадресації -> Словники
- Правила переадресації -> Матеріали -> Назва правила
- Правила переадресації -> Словники -> Назва правила
- Правила переадресації -> Матеріали -> Додати правило
- Правила переадресації -> Словники -> Додати правило

Що приведено до продакшен-стану
-------------------------------
- окремий breadcrumb builder не використовується;
- логіка хлібних крихт зосереджена в одному місці через hook_system_breadcrumb_alter();
- прибрано зайвий рівень bundle у хлібних крихтах;
- прибрано додаткові крихти дій на кшталт видалення, увімкнення та вимкнення;
- форма додавання правила коректно визначає розділ через query-параметри entity_type і bundle;
- підтвердження видалення повертає до списку правил поточного набору;
- описи модуля, permissions, schema та службові тексти приведені до однорідної української мови.
